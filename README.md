## 📂 디렉터리 구조
```
/ ── network/
    ├── book/              # 책/강의 정리
    ├── exercises/         # 실습 코드 및 정리
    ├── resources/         # 참고 자료 및 링크
    ├── README.md          # 저장소 개요
```

## 🌱 브랜치 전략
main: 최종 정리본 (수동 병합)
excercises/{본인이름}
book/{본인이름}
main에 병합시 브랜치 삭제

3. ✏️ 커밋 메시지 규칙

[Book] 책 / 강의 요약 작성

[Exercise] 실습 코드 작성

[Assignment] 과제 수행

[Fix] 오류 수정

[Docs] 문서 정리

[Refactor] 코드 리팩토링

예시
- [Lecture] 1.1 컴퓨터 네트워크와 인터넷
- [Exercise] TCP 연결 실습 추가
- [Fix] 패킷 캡처 코드 오류 수정
